# CJ's Portfolio

This is my personal portfolio website, built with HTML and CSS, hosted on **GitHub Pages** and served through **Cloudflare** for enhanced performance and security.

## Features
- Minimalist design with easy navigation.
- Links to my **GitHub, resume, LinkedIn, and blog**.
- Dedicated sections for **notes, tech blogs, and courses I've taken**.

## Hosting Setup
1. **GitHub Pages** hosts the site.
2. **Cloudflare** manages DNS and security.
